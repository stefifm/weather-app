# Weather App

This is a simple weather app that uses the OpenWeatherMap API to get the current weather for a given city.
Base on tutorial from [freeCodeCamp](https://www.youtube.com/watch?v=KkC_wYM_Co4).

## Technologies
- Next.js 14
- Tailwind CSS
- OpenWeatherMap API
- TypeScript
- Jotai
- React Query
- React Icons
- Axios
